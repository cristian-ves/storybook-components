# React UI Components

A small React component library built with **TypeScript**, documented using **Storybook**, and published on **npm**.

## Components

-   Button
-   Select
-   MyLabel

## Installation

```bash
npm i cav-custom-components
```

## Storybook

Run Storybook locally:

```bash
npm run storybook
```

## Tech Stack

-   React
-   TypeScript
-   Storybook
-   Vite
